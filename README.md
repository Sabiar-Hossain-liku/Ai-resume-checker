# Ai Resume Evaluator
This project is a full-stack AI-powered application.

## Features
- User authentication and registration
- AI-driven interview report generation
- Secure REST API integration
- Modern, responsive frontend UI

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: React, Vite, SCSS

## Folder Structure
- Backend/: Express server, API, AI services, authentication
- Frontend/: React app, routes, features, styles

## Setup
1. Install dependencies in both Backend and Frontend folders
2. Start the backend server
3. Start the frontend development server
